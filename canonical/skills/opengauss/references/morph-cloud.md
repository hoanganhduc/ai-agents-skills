# OpenGauss Morph cloud template

Upstream hosts an instant-start Morph template:

- https://morph.new/opengauss-0-2-2

This is a **manual** remote path. AAS `config-snippet` may print the URL with placeholders only.

- Never auto-launch Morph from installer smoke
- Never store Morph credentials in the repo
- Treat Morph/OpenGauss output under the same evidence policy as local runs (`opengauss_run` ≠ claim support)
