# Report Contract

`recommendation.md` must include:

- scope and inspected inputs
- delivery status
- ranked venue shortlist
- per-venue evidence summary
- score components with caveats
- unsupported or stale evidence gaps
- Review Findings
- Delivery Check

Separate observations from inferred fit. Include `incomplete analysis` when
material evidence is missing or provider coverage is blocked.

Do not include raw draft text by default.
